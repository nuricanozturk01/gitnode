Git hosting, CI/CD Actions, enterprise SSO, admin tooling, observability, and audit — a cohesive experience for teams that want GitHub-like workflows without the SaaS footprint.

Public and private bare repositories on your own disks — with the same owner/repo layout you expect from hosted Git.

Branch management

HTTP & HTTPS Git

Smart Git over the same origin as your API (/git/…). Clone, fetch, and push with your account via HTTP Basic — TLS-ready when you terminate HTTPS in front.

JGit Smart HTTP Basic auth

SSH Git

Native Git over SSH with deploy keys and per-account keys. Works with standard git@host:repo URLs and your ~/.ssh/config aliases.

JGit MINA SSHD ED25519

Project management

Kanban boards with tasks and subtasks, linked to repository branches and pull requests. Create work branches from the board, track open PRs on cards, and optionally complete items when a PR merges — tuned per project in settings. Projects linked to a repository are browseable and paginated directly from the repo's Projects tab.

Available after sign-in — open Projects from the app bar.

Includes
Boards & columns Tasks & subtasks Branch & PR linking Merge automation

Code snippets

Gist-style shares for one-off scripts and notes — revisioned, syntax-highlighted, and scoped to who can see them. Browse paginated snippets per repository or across all public ones.

Available after sign-in — open Snippets from the app bar.

Public & private Revision history Multi-file Fork

Issues

Track bugs and feature requests per repository with labels, comments, and status. Issues can be linked to tasks on a Kanban board to keep planning in sync with the codebase.

Available in the Issues tab of any repository.

Labels Comments Task linking

Collaborators

Invite team members to your repositories with fine-grained permissions: push access, PR management, issue control, settings, or full admin. Share an invite link with a configurable expiry.

Fine-grained perms Invite links Admin role

OriginHub Actions

Event-driven CI/CD on your own infrastructure. Define workflows in YAML, dispatch jobs to self-hosted runners over WebSocket, stream real-time logs via SSE, and store artifacts and cache without leaving your environment. Secrets are encrypted at rest with AES-256 and scoped per repository.

Available in the Actions tab of any repository.

Includes
YAML workflows Self-hosted runners AES-256 secrets Artifact store Cache store SSE log streaming

SAML & LDAP SSO

Per-organization identity: map email domains to SAML 2.0 IdPs or corporate LDAP directories. Users sign in with work email from the login page; JIT provisioning creates accounts on first successful authentication. Configure metadata, connection tests, and directory search bases from the admin panel.

Includes
SAML 2.0 SP LDAP bind Domain routing Group mapping

Admin panel

Separate platform-admin UI for instance statistics, user enable/disable, and organization lifecycle — including SAML and LDAP configuration per org.

Dashboard stats User management Org SSO/LDAP

Rate limiting

Redis-backed request limits on sensitive endpoints — login, registration, repo creation, webhooks, and SSO discovery — to reduce abuse and credential stuffing.

Per-endpoint keys Redis sliding window

Prometheus & Grafana

Spring Boot Actuator exposes Micrometer metrics at /actuator/prometheus. Docker Compose ships Prometheus and a pre-provisioned Grafana dashboard for JVM, HTTP, and custom tags.

Micrometer Grafana dashboard

Audit logging

Application-level audit trail with partitioned storage, append-only enforcement, and admin API queries. PostgreSQL ships with pgAudit for database write, DDL, and role events in server logs.

@Audited actions pgAudit Admin API
